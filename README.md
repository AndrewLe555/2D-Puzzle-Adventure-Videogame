# 2D-Puzzle-Adventure-Videogame

Project Overview:
A simple puzzle video game made using the python library Pygame. Features a math puzzle, shape puzzle, obstacle course, and an enemy puzzle. The game is designed to make fun of you for solving the puzzles. 

Resources used:

References to Monty Python, The Legened of Zelda, Tommy Tutone, and Bob Ross

Project Purpose:
Introduced concepts such as level design, game loop, and the Pygame library. Came across many problems in development, such as collision with the walls, movement of the character/ flipping the character image, and general game logic. Learned how to debug, researc documentation, and perform trial and error, and more to overcome these problems. 

Technologies Used: Python and Pygame library

Setup:
1. Install pygame: https://www.pygame.org/wiki/GettingStarted
2. Make sure all files are in the same folder
3. Perform the command python simple_puzzle_game.py

Key Features/Functionalities:
This project features a playable character, who moves using w, a, s, and d. The player learns about an ancient civilization through a series of notes lying around. The player attempts to complete 4 puzzles which include a math puzzle, a shape puzzle, an obstacle course, and an enemy puzzle. Then you have to solve one final puzzle to complete the game. Then, once you do this, the main game loop ends and the finale begins!

Challenges/Reflection:
The main challenge with this project was getting started. This was my first personal project, and I had little idea of what I was doing. I did lots of research on youtube and pygame documentation, to learn what was out there. Then, I had to think of what type of game to make. Originally, I wanted to make a platformer, but the physics were very difficult to implement. So, I pivoted to a 2D puzzle game. It was a fun challenge and it really helped teach me to not give up on your code, even when you believe it is right.
